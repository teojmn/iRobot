Le Projet

Bienvenue sur la page du projet. Voici quelques informations clés :

Objectif du Projet

Ce projet vise à fournir une solution de gestion des casiers et des emprunts.

Fonctionnalités

  • Gestion des casiers
  • Gestion des emprunts
  • Interface utilisateur intuitive
  • Support pour les utilisateurs

Technologies Utilisées

Ce projet utilise Flask pour le backend et Vue.js pour le frontend.
